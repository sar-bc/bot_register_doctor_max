# 🤖 Бот поликлиники для MAX

Бот для вызова врача на дом. Разработан для мессенджера MAX.

## 📋 Функционал

### Для пациентов

- 👤 Регистрация и авторизация
- 📝 Вызов врача на дом (анкета из 9 шагов)
- 📋 Просмотр истории вызовов
- ❌ Отмена вызова (только за сегодняшний день)
- 🏠 Главное меню с кнопками

### Для регистраторов (role=1)

- 🔔 Получение уведомлений о новых вызовах
- ✅ Подтверждение/отклонение вызовов
- 👨‍⚕️ Назначение врачей на вызовы
- 📊 Просмотр статистики

### Для администраторов (role=0)

- 🛠️ Админ-панель с управлением
- 👨‍⚕️ Управление врачами (CRUD)
- ⏰ Настройка расписания работы бота
- 📊 Статистика и экспорт в Excel
- 👥 Управление регистраторами

## 🚀 Установка и запуск

### Локальный запуск (venv)

```bash
# 1. Клонируем репозиторий
git clone https://github.com/sar-bc/bot_register_doctor_max.git
cd bot_register_doctor_max

# 2. Создаем виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# 3. Устанавливаем зависимости
pip install -r bot/requirements.txt

# 4. Настраиваем .env файл
cp .env_template .env
# Отредактируйте .env: добавьте токен бота и настройки БД

# 5. Запускаем бота
cd bot
python main.py
```

### Запуск через Docker

```bash
# 1. Клонируем репозиторий
git clone https://github.com/sar-bc/bot_register_doctor_max.git
cd bot_register_doctor_max

# 2. Настраиваем .env файл
cp .env_template .env
# Отредактируйте .env: добавьте токен бота и настройки БД

# 3. Запускаем контейнеры
docker-compose up -d

# 4. Смотрим логи
docker-compose logs -f bot_doctor_max
```

### 📝 Команды

Для всех пользователей
/start - начать работу

/help - справка

/call - вызвать врача

/my_calls - мои вызовы

/cancel_call - отменить вызов

#id - узнать свой ID

### Для администраторов

/admin - открыть админ-панель

/admin_help - справка по администрированию

#addadmin ID - добавить администратора

#deladmin ID - удалить администратора

#admins - список администраторов

#addstaff ID - добавить регистратора

#delstaff ID - удалить регистратора

#staff - список регистраторов

### 🏗️ Структура проекта

bot_register_doctor_max/
├── bot/                    # Основной код бота
│   ├── app/               # Обработчики и клавиатуры
│   ├── core/              # Логирование и конфиги
│   ├── database/          # Модели и работа с БД
│   ├── main.py            # Точка входа
│   └── requirements.txt   # Зависимости
├── db/                    # SQL скрипты для БД
├── data/                  # Данные MySQL
├── temp/                  # Временные файлы
├── docker-compose.yml     # Docker compose
├── .env_template          # Шаблон переменных окружения
└── README.md              # Этот файл

### 🐳 Переменные окружения (.env)

```env
MAX_BOT_TOKEN=your_bot_token
MYSQL_HOST=22.0.1.9
MYSQL_DATABASE=bot_doctor
MYSQL_USER=your_user
MYSQL_PASSWORD=your_password
```

### 📊 Статистика и отчеты

Просмотр статистики за день

Экспорт данных в Excel за любой период

Автоматическая нумерация вызовов

👥 Роли пользователей
Роль	Значение	Описание
0	Администратор	Полный доступ, управление ботом
1	Регистратор	Прием/отклонение вызовов
(нет)	Пациент	Только вызов врача
